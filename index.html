LESS is more

Ett effektivare sätt att styla HTML

LESS

LESS is a dynamic stylesheet language and a CSS pre-processor

LESS i praktiken

Du skriver LESS-kod för att styla dina HTML-element

Syntaxen är väldigt lik CSS

All CSS-kod är giltig i en LESS-fil

Koden skrivs i en fil med ändelsen .less

Filen kompileras

Kompilatorn skapar automatiskt en CSS-fil

Fördelarna med LESS

Variabler


					@lightBlue:#ddf;
 
					.shape{
					    background:@lightBlue;
					}

Variabler


					@lightRed:   #fdd;
					@lightGreen: #dfd;
					@lightBlue:  #ddf;
 
					@defaultThemeColor:@lightGreen;
 
					.shape{
				    background:@defaultThemeColor;
					}

Mixins

        		
        			.Round {
					    -webkit-border-radius: 9999px; 
					    -moz-border-radius: 9999px; 
					    border-radius: 9999px;
					}
					.shape {
					    .Round
					}

Matematiska operationer


					@baseFontSize: 16px;
					
					p {
					    font-size: @baseFontSize + 4;
					}

Färgfunktioner


					darken() och lighten(), som lägger på lite svart eller vitt
